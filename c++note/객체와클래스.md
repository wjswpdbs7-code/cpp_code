# 객체, 클래스


객체 클래스의 예시 코드
```cpp
#include <iostream>
#include <string>
using namespace std;
int main() {
    string str1 = "Hello";
    string str2 = "World";
    string str3 = str1 + str2; // 문자열 연결
    cout << str3; // HelloWorld 출력
    return 0;
}
```


1. 객체: 객체는 클래스의 인스턴스이다. 클래스는 객체를 생성하기 위한 청사진 또는 템플릿으로, 객체는 클래스에서 정의된 속성과 행동을 가진 구체적인 실체이다. 예시에서는 str3가 객체이다.

2. 클래스: 클래스는 객체를 생성하기 위한 설계도 또는 템플릿이다. 클래스는 속성과 행동을 정의하며, 객체는 이러한 속성과 행동을 가진 구체적인 실체이다. 예시에서는 string이 클래스이다.
